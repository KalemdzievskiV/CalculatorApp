# MDA Калкулатор — Modular Design Architects

Функционална веб-апликација за пресметка на цена на модуларна куќа, изградена по
методологијата од `PRESMETKA 26.03.2026.xlsx`. Целата пресметка (материјали, цени,
формули за количини, влезни параметри) е уредлива од админ панел.

## Стартување

Нема зависности — потребен е само Node.js (≥16):

```bash
cd app
node server.js
```

- **Калкулатор:** http://localhost:3000
- **Админ панел:** http://localhost:3000/admin.html — лозинка `admin123`
- Порта: `PORT=8080 node server.js`

## Што има внатре

| Дел | Опис |
|---|---|
| Лендинг (`/`) | Целосна страница по „Direction A" дизајнот: херо, маркиза, „Зошто MDA" со бројачи, модели на куќи (со реални „од" цени пресметани од моторот — клик на модел го пополнува калкулаторот), калкулатор и контакт. |
| Калкулатор — визард во 4 чекори | **1. Површина и систем** (лизгачи за квадратура/тераса, конструктивен систем, прецизни мерки со рачно прегазување), **2. Клучни ставки** (покрив, фасада, внатрешно, плафон), **3. Инсталации и финиш** (електрика, бањи, врати, подови, дограма со автоматски скалиран сет + рачно уредлива листа), **4. Резиме** (конфигурација, сива фаза/турнкеј, попуст, печатена понуда, зачувување, детална спецификација по материјали). Секогаш видлив резиме-панел со анимирана вкупна цена, цена/m² и сива фаза. Мерките (ѕидови, кров, симс) автоматски се изведуваат од квадратурата, пропорционално на референтната куќа од Excel-от. |
| Админ (`/admin.html`) | CRUD за **материјали** (име, категорија, ед. мера, цена, формула за количина со валидација и тест-пресметка во живо, фактор за сива фаза, вкл./искл.), **категории** (додавање, преименување, редослед), **влезни параметри** (нови полиња во калкулаторот), **стандардна дограма**, преглед/бришење **понуди**, **подесувања** (ДДВ, курс, лозинка). |

## Формули

Количината на секој материјал е израз кој користи:

- влезни параметри: `AREA`, `AREA_HOUSE`, `WALL_OUT`, `WALL_IN`, `FLOOR_H`, `ROOF`, `WC`, `KAMENA`, `SIP`, `ELEKTRIKA`… (да/не параметрите се 0/1)
- количини од други ставки: `qty("K01")`
- износи во МКД од други ставки: `total("EM01")`
- оператори `+ - * / ( )` и функции `min max round ceil floor abs`

Пример (стиропор за фасада): `((FLOOR_H*WALL_OUT)+5)*ABRIB`

## Архитектура

```
app/
├── server.js          — чист Node HTTP сервер: статика + REST API + токен автентикација
├── data/seed.js       — почетна база: ~150 материјали/формули извлечени од Excel
├── data/db.json       — JSON база (се создава при прв старт, ги чува сите измени)
├── public/
│   ├── index.html     — јавен калкулатор
│   ├── admin.html     — админ панел
│   ├── js/engine.js   — пресметковен мотор (безбеден парсер на формули, работи и во Node и во прелистувач)
│   ├── js/app.js      — логика на калкулаторот
│   ├── js/admin.js    — логика на админ панелот
│   └── css/style.css  — MDA дизајн систем (Direction A)
└── test/verify.js     — верификација наспроти бројките од Excel
```

## Верификација

```bash
node test/verify.js
```

Со стандардните параметри апликацијата ги репродуцира точно бројките од
оригиналниот Excel: **112.640,34 € вкупно**, 17.182,42 € ДДВ, 631,36 €/m²,
сива фаза 67.520,13 €, плус сите 12 категориски збирови од листот RECAPITLAR.

За ресетирање на базата кон почетните податоци: избришете `data/db.json` и рестартирајте.
